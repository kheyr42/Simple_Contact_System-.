# Simple Contact Management System

## Description
This project is a Simple Contact Management System developed using PHP and MySQL.
It allows users to register, login, manage contacts, and perform CRUD operations
(Create, Read, Update, Delete) securely using sessions.

This project was developed as part of the Web Application Development course.

---

## Features
- User Login & Logout
- Session Management
- Dashboard
- Add New Contact
- View Contacts
- Edit Contact
- Delete Contact
- Session Expiration

---

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- XAMPP Server

1. Open phpMyAdmin:

http://localhost/phpmyadmin

2. Create a database named:

simple_contact_db

3. Import the SQL file:
- Go to Import
- Select `users.sql`
- Click Go
